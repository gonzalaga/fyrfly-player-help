Common questions

These answers cover device and software support. Shipping, returns, refunds, order status, and store-policy questions are handled separately by customer support.

What should I do if Spotify will not let me sign in on a Fyrfly Player?

Connect the player to Wi-Fi, make sure Spotify is updated from the Fyrfly or Campfire app store, then restart the device and try the login again.

If the login opens a web flow or loops back to the same screen, contact support with the device model, OS version, Spotify version, and a photo of the screen where it gets stuck.

Spotify Login Apps
How do I update or reinstall apps on the player?

Open the Fyrfly or Campfire app store while connected to Wi-Fi, check for the latest available app version, and install the update from there.

If an app is missing or fails to update, restart the player and try again. If it still fails, contact support with the app name and current version.

App store Updates Downloads
What should I do if the player will not charge or power on?

Try a known-good USB cable and wall charger, leave the player connected for at least 30 minutes, then hold the power button for a full restart.

If it still will not power on, send support the model, charging light behavior, and whether the screen shows anything while plugged in.

Battery Charging Power
How do restrictions and parental controls work on the player?

The player is designed to limit access to approved apps and content. If a child can access something unexpected or a needed app is blocked, note the app name and where it appears.

When contacting support, include the steps that led there so the restriction setting involved can be identified quickly.

Restrictions Admin Access
What should I try if the player will not connect to Wi-Fi?

Restart the player, confirm the Wi-Fi password, and try a standard 2.4 GHz or 5 GHz network without a captive portal.

If the network uses a browser sign-in page, connect on a simpler network first or ask support for help with that network type.

Wi-Fi Network Setup
Where should I start when setting up a new Fyrfly Player?

Charge the player, connect it to Wi-Fi, complete the first-run setup screens, then open the app store to update core apps before handing it to a child.

If setup stops on a specific screen, send support a photo of that screen and the device model.

Setup Getting started App store
How do I add music or MP3 files to the player?

Use standard MP3 files and place them in the player music folder or supported storage location.

After copying files, safely eject the device or card and restart the music app so it can rescan the library.
